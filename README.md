# Weekly Paper Digest

Weekly Paper Digest は、ある研究分野における直近1週間の注目論文を収集し、音声で気軽に聞ける形へ変換するシステムです。

Weekly Paper Digest in Genome Microbiology 配信はこちら👉 https://kuroppy.github.io/weekly_paper_digest/

## Why?

研究の専門化は年々進んでいます。

同じ学会に所属していても、少し分野が離れるだけで研究内容を理解することが難しくなり、議論どころか質問をすることさえ難しい場面があります。

また、論文の出版速度は加速し続けており、研究者コミュニティの中で共有される知識や話題は相対的に少なくなっています。

研究者同士の議論は、共有された知識や話題の上に成り立っています。しかし近年は論文数の増加と専門分化によって、その「共有された話題」を持つこと自体が難しくなりつつあります。

Weekly Paper Digest は、この状況に対する小さな実験です。

専門から少し離れた研究も気軽に把握できるようにし、学会や研究会で異分野の研究者と会話するきっかけを作ることを目指しています。

また、新しい分野に興味を持った人が、その分野の最近の研究動向を素早く把握する入口として利用できることも期待しています。

## Concept

論文を深く理解することを目的としたプロジェクトではありません。

むしろ、

* 最近どのような研究が注目されているのか
* 分野の中でどのような流れが生まれているのか
* どのような研究者が何を議論しているのか

を気軽に把握するための「研究コミュニティのラジオ」のような存在を目指しています。

## Workflow

1. キーワード出現頻度とジャーナル重み付けによる論文スクリーニング
2. LLMによる注目トピック抽出
3. LLMによる重要論文選定
4. NotebookLMによる音声生成

## Customization

本プロジェクトは特定分野専用ではありません。

スクリーニングに利用するキーワードやジャーナル設定を変更することで、

* 微生物学
* 神経科学
* AI
* 生態学
* STS

など、様々な研究分野へ展開できます。

興味を持った方は自由に改変・再利用してください。

本リポジトリでは、作者の所属学会である”ゲノム微生物学”を例としたコードを公開しています。

## Disclaimer

本プロジェクトは生成AIを利用しています。

音声や要約内容について人による内容確認は行っておらず、正確性を保証するものではありません。

研究内容を正確に理解したい場合は、必ず原著論文をご確認ください。

本プロジェクトは研究動向に気軽に触れるための補助ツールとして利用してください。
