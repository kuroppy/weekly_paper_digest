# チャンネル説明
チャンネル名：weekly-genome-microbiology
リスナー対象：生物系の大学院生や研究者
チャンネルの目的：直近１週間のゲノム微生物学における重要論文を紹介する。
配信時間：15~20分

# 紹介の構成
- まずは、大きな今週の論文の大きな流れを３つのテーマで説明する。
- 各テーマで注目すべき論文（各３報）を紹介する。（各論文30秒程度で）
- 各テーマで深掘り紹介する論文（各１報）を、添付の論文pdfを参照して解説する。（各論文3-4分程度で）

# 添付ファイルの説明
- weekly_genome_microbiology.md：直近１週間のゲノム微生物学の注目論文情報（これが話の進行です）
- paper1.pdf：1つ目の深掘り論文です。
- paper2.pdf：2つ目の深掘り論文です。
- paper3.pdf：3つ目の深掘り論文です。

# 話のトーンや難易度
- 対象は微生物学を学んでいる大学院生や研究者です。
